# Code of Conduct

## 中文

AlphaPilot 是一个面向研究、教育和模拟盘观察的开源项目。我们希望这里像一个清晰、负责、
愿意互相帮忙的车队车库：可以提出问题、指出风险、改进工程，但不鼓励情绪化争吵、
夸大收益或误导他人进行真实交易。

## 我们鼓励的行为

- 尊重不同经验水平的贡献者，解释问题时尽量给出上下文和证据。
- 聚焦代码、文档、测试、复现步骤和风险边界。
- 在讨论交易、收益、回撤和模型表现时明确区分模拟盘、回测和真实交易。
- 报告问题时删除密钥、账户、服务器地址、日志中的隐私信息。
- 对安全、资金风险、公开控制面板、数据泄露等问题保持谨慎。

## 不接受的行为

- 人身攻击、骚扰、歧视、威胁或持续挑衅。
- 发布他人的隐私信息、凭证、账户截图或未脱敏日志。
- 将 AlphaPilot 宣称为投资建议、稳赚策略或真实交易保证。
- 引导他人绕过安全边界、公开控制接口或提交实盘凭证。
- 使用 issue、PR、讨论区进行广告、垃圾信息或无关推广。

## 处理方式

维护者可以删除不当内容、要求修改发言、关闭 issue/PR、限制互动，或在严重情况下封禁参与者。
如果你看到违反本准则的行为，请通过 GitHub 私下联系维护者，或使用 GitHub 的举报和安全建议功能。

## English

AlphaPilot is an open-source project for research, education, and paper-trading
observation. We want the community to feel like a careful and helpful racing
garage: discuss problems, share evidence, improve engineering, and stay clear
about risk boundaries.

Expected behavior:

- Be respectful and evidence-driven.
- Focus on code, docs, tests, reproduction steps, and operational risk.
- Clearly distinguish paper trading, backtests, and real-money trading.
- Redact secrets, account data, server details, and private logs.
- Treat security, financial automation, and public control surfaces with care.

Unacceptable behavior:

- Harassment, discrimination, threats, personal attacks, or persistent trolling.
- Publishing private credentials, account screenshots, or unredacted logs.
- Presenting AlphaPilot as investment advice, guaranteed profit, or a real-money
  trading promise.
- Encouraging others to bypass safety boundaries or expose control APIs.
- Spam, advertising, or unrelated promotion.

Maintainers may edit or remove inappropriate content, close issues or pull
requests, limit interactions, or block users for serious violations.
