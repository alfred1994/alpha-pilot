## No Performance Guarantee

Backtests, paper-trading results, dashboard values, LLM decisions, reports, and
sample outputs are not guarantees of future performance. Markets can change
quickly, data can be delayed or wrong, and automated systems can fail.

## Paper Trading Default

The intended default mode is:

```bash
BROKER_MODE=paper
```

Do not connect Quant Pilot to real-money execution unless you have independently
reviewed the code, risk controls, broker adapter, infrastructure, credentials,
monitoring, compliance requirements, and failure modes.

## LLM Limitations

LLM outputs can be incomplete, stale, hallucinated, overconfident, or malformed.
They must not be treated as reliable investment advice. Use human review and
independent validation for any serious financial decision.

## Public Dashboard

The public dashboard is an observability surface, not a control plane. It can show
paper account state, decisions, lessons, and system health, but it must not be used
as proof of profitability or as a solicitation to trade.
